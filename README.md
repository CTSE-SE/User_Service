# User Service - Food Ordering System

This project implements the User Service for a microservices-based Food Ordering System.

## Features
- User registration
- User login with JWT authentication
- Get logged-in user profile
- Update logged-in user profile
- Check whether a user exists for inter-service communication
- Health check endpoint
- Swagger/OpenAPI documentation
- Dockerized deployment

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- Swagger
- Docker
